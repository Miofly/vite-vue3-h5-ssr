# vite-vue3-ssr
vue3 H5端的ssr脚手架, 含asyncData里的xhr请求 (Vue3 + Vant + Pinia + Vite + TS + Unocss + SSR)

## Project setup
```bash
$ pnpm install
```

### Compiles and hot-reloads for development
```bash
$ pnpm serve
```

### Compiles and minifies for production
```bash
$ pnpm build
```

### Preview for production
```bash
$ pnpm start
```

### Lints and fixes files
```bash
$ pnpm lint
```
